## Experimental Scripts

This folder holds ad-hoc helpers that are still potentially useful during local debugging, but are not part of the supported package script surface.

- `dev/`: quick local launch helpers
- `inspection/`: one-off state inspection helpers
- `audit/`: exploratory audit scripts that emit artifacts under `output/`

Use these manually when needed. For supported validation and replay flows, prefer:

- `scripts/validation/`
- `scripts/replay/`
- `scripts/manual/`
