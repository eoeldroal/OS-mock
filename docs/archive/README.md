## Archived Documents

This directory contains historical reports, test summaries, and one-off validation artifacts that were previously stored at the repository root.

They are preserved for reference, but they are not the canonical description of the current system.

- `testing/batch6`: early batch-6 test campaign notes
- `testing/mail`: mail-specific validation notes
- `testing/popup`: popup and context-menu investigations
- `testing/sidebar`: file explorer sidebar investigations
- `testing/windowing`: drag, terminal workflow, and window-behavior reports
- `testing/round7`: later edge-case campaign reports
- `testing/general`: miscellaneous one-off JSON artifacts

For current architecture and operating guidance, prefer:

- `README.md`
- `README-Revised.md`
- `AGENTS.md`
- `docs/paper-draft/`
